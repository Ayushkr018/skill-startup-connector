API Overview

The SkillSync AI API allows you to integrate our intelligent talent matching platform into your applications. Our RESTful API provides endpoints for managing profiles, finding matches, and facilitating communication.

Base URL

https://api.skillsync.ai/v1

Response Format

All API responses are returned in JSON format with the following structure:

{ "success": true, "data": { ... }, "message": "Operation completed successfully", "timestamp": "2025-01-20T10:42:00Z" }

Authentication

SkillSync AI uses API keys for authentication. Include your API key in the Authorization header:

curl -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ https://api.skillsync.ai/v1/profiles

Getting Your API Key

To get your API key:

  1. Sign up for a SkillSync AI account
  2. Navigate to your dashboard
  3. Go to Settings â†’ API Keys
  4. Generate a new API key

API Endpoints

Matches API

GET

Get Matches

Retrieve matches for a specific user based on their profile and preferences.

GET /matches/:userId
Parameters
Parameter Type Required Description
userId string Required Unique user identifier
limit integer Optional Number of matches to return (default: 20)
min_score integer Optional Minimum match score (default: 60)
Example Response
{ "success": true, "data": { "matches": [ { "id": "match_123", "user_id": "user_456", "name": "TechStart Inc.", "type": "startup", "match_score": 92, "skills_matched": ["React", "Node.js", "Python"], "created_at": "2025-01-20T10:42:00Z" } ], "total": 15, "page": 1 } }
POST

Create Match Action

Accept or reject a match between users.

POST /matches/:matchId/action
Request Body
{ "action": "accept", // or "reject" "message": "I'm interested in this opportunity!" }

Profiles API

GET

Get Profile

Retrieve a user's profile information.

GET /profiles/:userId
PUT

Update Profile

Update user profile information.

PUT /profiles/:userId
Request Body
{ "name": "John Doe", "title": "Full Stack Developer", "skills": ["React", "Node.js", "Python"], "experience_years": 3, "location": "Remote", "bio": "Passionate developer with 3+ years experience..." }

Messages API

POST

Send Message

Send a message to another user.

POST /messages
Request Body
{ "recipient_id": "user_456", "message": "Hi! I'm interested in your startup.", "type": "text" }
GET

Get Conversations

Retrieve user's conversations.

GET /messages/conversations

Webhooks

SkillSync AI can send webhooks to notify your application of important events.

Supported Events

  • match.created - New match found
  • match.accepted - Match accepted by user
  • message.received - New message received
  • profile.updated - Profile information updated

Webhook Payload

{ "event": "match.created", "data": { "match_id": "match_123", "user_id": "user_456", "match_score": 92, "timestamp": "2025-01-20T10:42:00Z" }, "webhook_id": "webhook_789" }

SDKs & Libraries

Official SDKs and community libraries to integrate SkillSync AI into your applications.

JavaScript

Official JavaScript SDK for web and Node.js applications.

npm install @skillsync/api-client

Python

Python SDK for backend integrations and data analysis.

pip install skillsync-python

React

React components for easy frontend integration.

npm install @skillsync/react

Error Handling

SkillSync AI uses standard HTTP status codes and provides detailed error messages.

Common Error Codes

Status Code Meaning Description
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
403 Forbidden Access denied to resource
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred

Error Response Format

{ "success": false, "error": { "code": "INVALID_PARAMETERS", "message": "The userId parameter is required", "details": { "field": "userId", "issue": "required_field_missing" } }, "timestamp": "2025-01-20T10:42:00Z" }
